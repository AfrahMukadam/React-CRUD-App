## Responsive User Management CRUD Application


A responsive frontend CRUD application built with React.js using REST APIs.

## Features

- View all users
- Create user
- Edit user
- Delete user
- View user details
- Search users
- Responsive UI
- Form validation
- Loading states
- Error handling
- Reusable components

## Installation

### Clone repository

```bash
git clone YOUR_GITHUB_LINK
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm start
```
