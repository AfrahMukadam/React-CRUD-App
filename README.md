# User Management CRUD Application

A responsive frontend CRUD application built with React.js using REST APIs.

## Features

- View all users
- Create user
- Edit user
- Delete user
- View user details
- Search users
- Responsive UI
- Form validation
- Loading states
- Error handling
- Reusable components

## Tech Stack

- React.js
- Axios
- React Router DOM
- Bootstrap
- React Icons

## Folder Structure

src/
├── api
├── components
├── pages
├── App.js
└── App.css

## Installation

### Clone repository

```bash
git clone YOUR_GITHUB_LINK
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm start
```

## API Integration

Integrated with backend REST APIs using Axios.
